О Mozilla Thunderbird

Mozilla Thunderbird — это бесплатный, открытый и кроссплатформенный почтовый клиент, разработанный Mozilla Foundation. Программа обеспечивает удобную работу с электронной почтой, поддерживает множество протоколов и предоставляет мощные инструменты для организации корреспонденции.

Установка Thunderbird

Пошаговое руководство по загрузке и установке почтового клиента

Шаг 1: Загрузка программы

1

Переход на официальный сайт

Откройте страницу загрузки Thunderbird в любом браузере. Страница автоматически определит наилучшую версию для вашей операционной системы.

Официальная страница загрузки Mozilla Thunderbird (нажмите для увеличения)
2

Начало загрузки

Нажмите кнопку "Загрузить" для скачивания установочного файла. В зависимости от скорости интернета загрузка может занять несколько минут.

Файл будет автоматически сохранен в папку "Загрузки"
3

Сохранение файла

При появлении диалога сохранения выберите "Сохранить файл" и укажите папку для загрузки (рекомендуется папка "Загрузки").

Диалог сохранения установочного файла (нажмите для увеличения)

Шаг 2: Установка программы

1

Запуск установки

Найдите скачанный файл в папке "Загрузки" и запустите его двойным щелчком мыши.

Установочный файл в папке загрузок (нажмите для увеличения)
2

Следование мастеру установки

Запустится мастер установки Thunderbird. Следуйте инструкциям на экране, оставляя настройки по умолчанию.

Мастер установки Mozilla Thunderbird (нажмите для увеличения)
3

Завершение установки

Дождитесь завершения процесса установки. После этого Thunderbird будет готов к настройке.

Поздравляем! Установка Thunderbird завершена успешно!

Настройка по протоколу IMAP

Рекомендуемый способ настройки с синхронизацией между устройствами

Преимущества протокола IMAP

При использовании протокола IMAP почтовая программа синхронизируется с сервером и сохраняет структуру папок вашего ящика. Отправляемые письма будут храниться не только на компьютере, но и на сервере, обеспечивая доступ с различных устройств.

Облачное хранение

Все письма хранятся на сервере

Мультиустройство

Доступ с любого устройства

Синхронизация

Автоматическая синхронизация папок

Создание учетной записи

1

Добавление новой учетной записи

Если у вас уже есть настроенная учетная запись, в верхнем меню выберите: Файл → Создать → Настроить мою учетную запись почты...

Меню создания новой учетной записи почты (нажмите для увеличения)
2

Заполнение основных данных

В окне "Настройка учётной записи почты" укажите следующие параметры:

Ваше имя: Имя пользователя (например, "Валентина Терешкова")
Адрес эл. почты: Ваш почтовый адрес (например, v.tereshkova@tez-tour.com)
Пароль: Ваш пароль от почтового ящика
Форма ввода данных учетной записи (нажмите для увеличения)
3

Ручная настройка

Не дожидаясь окончания автоматической настройки, нажмите кнопку Настройка вручную для указания точных параметров сервера:

Переход к ручной настройке параметров (нажмите для увеличения)

Настройка серверов

Входящая почта (IMAP)

Протокол: IMAP
Имя сервера: imap.tez-tour.com
Порт: 993
Шифрование: SSL
Аутентификация: Обычный пароль

Исходящая почта (SMTP)

Имя сервера: smtp.tez-tour.com
Порт: 465
Шифрование: SSL
Аутентификация: Обычный пароль
Имя пользователя:

Для домена tez-tour.com используйте только логин: v.tereshkova

Для других доменов (например, tez-tour.at) используйте полный адрес: username@tez-tour.at

Настройки IMAP сервера в Thunderbird (нажмите для увеличения)

1

Доступ к параметрам

Для доступа к настройкам учетной записи используйте один из способов:

Настройка по протоколу POP3

Альтернативный способ настройки с локальным хранением писем

Особенности протокола POP3

При использовании протокола POP3 все письма загружаются и сохраняются на ваш компьютер. Ключевое отличие от IMAP заключается в том, что письма хранятся локально, а не на сервере.

IMAP
  • Письма хранятся на сервере
  • Доступ с любого устройства
  • Синхронизация папок
  • Требует интернет для чтения
POP3
  • Письма хранятся локально
  • Доступ только с одного устройства
  • Быстрый доступ к письмам
  • Работа без интернета

Настройка серверов POP3

Шаги 1-4 выполняются аналогично настройке IMAP. Различия начинаются с настройки серверов:

Входящая почта (POP3)

Протокол: POP
Имя сервера: pop.tez-tour.com
Порт: 995
Шифрование: SSL
Аутентификация: Обычный пароль

Исходящая почта (SMTP)

Имя сервера: smtp.tez-tour.com
Порт: 465
Шифрование: SSL
Аутентификация: Обычный пароль
Имя пользователя:

Для домена tez-tour.com: v.tereshkova

Для других доменов: username@tez-tour.at

Настройки POP3 сервера в Thunderbird (нажмите для увеличения)

Нажмите Перетестировать для проверки настроек, затем Готово при успешной проверке.

Дополнительные настройки POP3

1

Настройка хранения сообщений

В разделе Параметры сервера снимите флажок с пункта Оставлять копии сообщений на сервере.

Важно:

При снятии этой галочки письма будут удаляться с сервера после загрузки в почтовую программу. Это освобождает место на сервере, но делает письма недоступными с других устройств.

Настройка хранения копий сообщений для POP3 (нажмите для увеличения)
2

Завершение настройки

Нажмите OK для применения изменений и перезапустите программу.

Настройка учетной записи POP3 завершена!
Дополнительные настройки:

Для изменения дополнительных параметров выберите в меню Инструменты → Параметры учетной записи.

Доступ к дополнительным настройкам через меню (нажмите для увеличения)
