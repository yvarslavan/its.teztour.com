Что происходит?

Сайты не загружаются и появляются различные сообщения об ошибках. Эта проблема может затрагивать любые веб-ресурсы и проявляться по-разному.

Типичные сообщения об ошибках:

Не удается найти DNS-адрес сервера
Не удается получить доступ к сайту
Превышено время ожидания
Страница временно недоступна
Соединение сброшено
Пример ошибки доступа к веб-сайту

Обновление кэша DNS

Очистка DNS-кэша часто решает проблемы с доступом к сайтам

Команда для очистки DNS

1

Откройте командную строку от имени администратора

Нажмите Пуск, введите cmd или Командная строка, щелкните правой кнопкой мыши и выберите «Запуск от имени администратора».

Запуск командной строки с правами администратора
2

Выполните команду очистки DNS

Введите команду ipconfig/flushdns и нажмите Enter.

ipconfig/flushdns
Результат выполнения команды очистки DNS
DNS-кэш успешно очищен!

Перезапуск службы DNS

Проверка и настройка службы DNS-клиента в Windows

Настройка службы DNS-клиента

1

Откройте службы Windows

Нажмите Пуск, введите services.msc и нажмите Enter.

services.msc
2

Настройте службу DNS-клиент

Найдите «DNS-клиент», дважды щелкните по ней и проверьте настройки:

  • Состояние: должно быть «Работает»
  • Тип запуска: должен быть «Автоматически»
Окно настроек службы DNS-клиента
3

Перезапустите службу

Если служба не работает, нажмите кнопку «Перезапустить» в верхней части окна служб.

Настройка Google DNS

Использование надежных DNS-серверов Google для улучшения соединения

Установка DNS от Google

1

Откройте сетевые подключения

Нажмите Пуск, введите ncpa.cpl и нажмите Enter.

ncpa.cpl
Выполнение команды для открытия сетевых подключений
2

Выберите активное подключение

Найдите активное интернет-подключение, щелкните правой кнопкой мыши и выберите «Свойства».

3

Настройте DNS-серверы

Дважды щелкните на «Протокол Интернета версии 4 (TCP/IPv4)» и введите DNS-адреса Google:

Предпочитаемый DNS: 8.8.8.8
Альтернативный DNS: 8.8.4.4
Ввод DNS-адресов Google в настройках сети
4

Сохраните настройки

Нажмите «ОК» для сохранения изменений и перезагрузите компьютер.

Очистка кэша браузера

Удаление временных файлов для всех популярных браузеров

Универсальный способ

Для любого браузера используйте комбинацию клавиш:

Ctrl + Shift + Del
Универсальная комбинация для очистки кэша
Chrome
Firefox
Edge/IE
Opera

Google Chrome

1 Нажмите Ctrl + Shift + Del
2 Выберите период «За все время»
3 Отметьте: Файлы cookie и Кэшированные файлы
4 Нажмите «Удалить данные»
Окно очистки данных в Google Chrome

Mozilla Firefox

1 Нажмите Ctrl + Shift + Del
2 Нажмите «Подробности»
3 Выберите период «Все»
4 Оставьте: Куки и Кэш
5 Нажмите «Удалить сейчас»
Окно очистки истории в Mozilla Firefox

Microsoft Edge / Internet Explorer

1 Нажмите Ctrl + Shift + Del
2 Отметьте: Временные файлы и Файлы cookie
3 Нажмите «Удалить»
Окно удаления данных в Internet Explorer

Opera

1 Нажмите Ctrl + Shift + Del
2 Выберите период «С самого начала»
3 Отметьте: Файлы cookie и Кэшированные файлы
4 Нажмите «Очистить историю»
Окно очистки данных в Opera
