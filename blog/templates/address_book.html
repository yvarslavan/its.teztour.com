О корпоративной адресной книге

Корпоративная адресная книга — это централизованная система контактов, которая автоматически синхронизируется с ERP-системой компании. Она позволяет быстро находить email-адреса сотрудников и автоматически дополнять адреса при написании писем.

Быстрый поиск контактов
Автоматическая синхронизация
Автодополнение адресов
Интеграция с ERP
Важно знать:

Для подключения к адресной книге потребуется пароль, который будет предоставлен IT-отделом. Убедитесь, что у вас есть доступ к корпоративной сети.

Настройка Mozilla Thunderbird

Пошаговое руководство по подключению к LDAP-серверу адресной книги

Открытие адресной книги

1

Запуск Thunderbird

Откройте Mozilla Thunderbird на вашем компьютере. Убедитесь, что программа полностью загружена и готова к работе.

Совет:

Если Thunderbird не установлен, скачайте его с официального сайта Mozilla

2

Переход к адресной книге

В главном окне программы нажмите кнопку Адресная книга на панели инструментов. Откроется окно управления контактами.

Меню "Новая адресная книга" → "Добавить адресную книгу LDAP" (нажмите для увеличения)
Где найти:

Кнопка "Новая адресная книга" находится в левой панели адресной книги Thunderbird

Создание LDAP-каталога

1

Создание нового каталога

В окне «Адресная книга» выберите меню Новая адресная книга → Добавить адресную книгу LDAP для добавления корпоративной адресной книги.

Новая адресная книга
Добавить адресную книгу LDAP
Альтернативный способ:

Можно также щелкнуть правой кнопкой мыши в области списка адресных книг и выбрать "Новый каталог LDAP"

Важно:

Убедитесь, что все параметры заполнены точно как указано выше. Неправильные настройки могут привести к ошибкам подключения

1

Сохранение настроек

После заполнения всех полей нажмите кнопку OK для сохранения конфигурации.

Новая адресная книга TEZ TOUR появится в списке адресных книг
Проверка подключения:

После сохранения настроек попробуйте выполнить поиск в адресной книге, чтобы убедиться в правильности конфигурации

Основные настройки LDAP-сервера: название, сервер, корневой элемент (нажмите для увеличения)

Основные настройки

Заполните поля на вкладке «Основное» следующими параметрами для подключения к корпоративному LDAP-серверу:

Параметры сервера

Название: Любое имя адресной книги (например, "TEZ TOUR")
Имя сервера: noodle.tez-tour.com
Корневой элемент (Base DN): dc=moscow,dc=teztour,dc=com
Порт: 389
Имя пользователя (Bind DN): cn=ldap,ou=users,dc=moscow,dc=teztour,dc=com
Пароль: Предоставляется IT-отделом
Настройки по умолчанию:

Указанные параметры являются оптимальными для большинства корпоративных LDAP-серверов

1

Сохранение настроек

После заполнения всех полей нажмите кнопку OK для сохранения конфигурации.

Новая адресная книга TEZ TOUR появится в списке адресных книг
Проверка настроек:

После сохранения настроек новая адресная книга "TEZ TOUR" должна появиться в левой панели окна адресной книги

Адресная книга TEZ TOUR в списке доступных каталогов с поиском по символу "@" (нажмите для увеличения)

Дополнительные настройки

1

Переход к расширенным настройкам

Откройте вкладку «Дополнительно» в окне настроек LDAP-каталога для конфигурации параметров поиска и аутентификации.

Параметры поиска

Не возвращать более чем: 1000 результатов
Область поиска: Поддерево
Фильтр поиска: (objectclass=*)
Способ аутентификации: Простой
Дополнительные настройки:

На вкладке "Дополнительно" можно настроить параметры поиска и аутентификации для оптимальной работы с LDAP-сервером

Рекомендуемые настройки:

Эти параметры обеспечивают оптимальную производительность и совместимость с большинством LDAP-серверов

2

Применение настроек

Нажмите кнопку OK для сохранения дополнительных параметров и закройте окно адресной книги.

Вкладка "Дополнительно" с параметрами поиска и аутентификации (нажмите для увеличения)

Настройка автодополнения

1

Переход к настройкам составления

Откройте Инструменты → Настройки → Составление → вкладка Адресация для настройки автоматического поиска контактов.

Инструменты
Настройки
Составление
Адресация
2

Включение поиска в каталоге

В пункте При вводе адреса искать подходящие почтовые адреса в: отметьте Сервере каталогов и выберите созданный LDAP-сервер.

После включения этой опции Thunderbird будет автоматически искать контакты в корпоративной адресной книге
3

Перезапуск программы

Перезапустите Mozilla Thunderbird для применения всех настроек автодополнения адресов.

Важно:

Перезапуск необходим для корректной работы автодополнения из LDAP-каталога

Настройки автодополнения адресов в разделе "Составление" → "Адресация" (нажмите для увеличения)
Автоматическое дополнение email-адресов из LDAP-каталога при написании письма (нажмите для увеличения)

Использование адресной книги

Практические примеры работы с корпоративными контактами

Поиск контактов

После настройки вы сможете быстро находить контакты сотрудников и использовать автодополнение при написании писем.

Автодополнение при вводе

Начните вводить имя или email сотрудника в поле "Кому" при создании письма

Просмотр всех контактов

Введите символ "@" в строке поиска адресной книги для отображения всех сотрудников

Фильтрация по критериям

Используйте различные поисковые запросы для нахождения нужных контактов

1

Поиск всех сотрудников

Для просмотра полного списка адресов откройте адресную книгу, выберите каталог TEZ TOUR и введите в строке поиска символ @.

Как это работает:

Символ "@" является универсальным поисковым запросом, который покажет все email-адреса в каталоге

2

Автодополнение при написании

При создании нового письма начните вводить фамилию сотрудника в поле адресата. Thunderbird автоматически предложит подходящие варианты из корпоративной адресной книги.

Теперь поиск контактов стал быстрым и удобным!

Настройка карточек контактов

1

Доступ к настройкам

В окне «Адресная книга» выберите меню Инструменты → Настройки (или Правка → Настройки в зависимости от версии).

Расположение меню:

В разных версиях Thunderbird меню настроек может находиться в разных местах. Ищите пункт "Настройки" или "Параметры"

2

Настройка автодополнения

В открывшемся окне «Настройки» перейдите на вкладку Адресация и настройте параметры автоматического дополнения адресов.

3

Выбор источника контактов

В разделе «Автодополнение адресов» отметьте галочкой Сервер каталогов и выберите из выпадающего списка созданную ранее адресную книгу «TEZ TOUR».

Внимание:

Обязательно выберите именно "TEZ TOUR" из списка, а не другую адресную книгу

4

Сохранение настроек

Нажмите OK для сохранения всех настроек автодополнения адресов.

Готово!

Корпоративная адресная книга настроена и готова к использованию
