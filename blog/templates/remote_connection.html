Метод 1: AnyDesk

Рекомендуемое решение для удаленного подключения

Что такое AnyDesk?

AnyDesk — это бесплатная программа удаленного доступа, которая позволяет безопасно подключаться к компьютеру через интернет. Программа поддерживает все популярные операционные системы: Windows, macOS, Linux, Android и другие.

Шаг 2: Настройка неконтролируемого доступа

Важно для безопасности:

Настройка неконтролируемого доступа позволит специалисту подключаться к вашему компьютеру без подтверждения. Обязательно используйте сложный пароль и сообщайте его только проверенным специалистам поддержки.

1

Откройте настройки

В главном окне AnyDesk нажмите на кнопку «Общие», затем выберите «Настройки».

Переход в настройки AnyDesk
2

Перейдите в раздел безопасности

В меню настроек выберите раздел «Безопасность».

Раздел настроек безопасности
3

Разблокируйте настройки безопасности

Нажмите красную кнопку «Разблокировать настройки безопасности». Это необходимо для получения прав администратора.

Подтверждение прав администратора в Windows
4

Активируйте неконтролируемый доступ

После разблокировки станет доступна опция «Разрешить неконтролируемый доступ». Установите флажок для активации.

Активация неконтролируемого доступа
5

Установите пароль

Система потребует установить пароль для безопасного доступа. Используйте сложный пароль и обязательно запомните его.

Установка пароля для удаленного доступа
Что сообщить специалисту:
  • ID вашего компьютера (отображается в главном окне AnyDesk)
  • Пароль, который вы установили

Отключение неконтролируемого доступа

Рекомендация по безопасности:

После решения проблемы рекомендуется отключить неконтролируемый доступ для обеспечения безопасности вашего компьютера.

1

Удалите токены авторизации

В настройках безопасности нажмите кнопку «Удалить токены авторизации». Это удалит все сохраненные подключения.

Удаление токенов авторизации
2

Измените пароль

Обязательно измените пароль доступа. Если оставить прежний пароль, возможно повторное автоматическое подключение.

Результат:

После выполнения этих действий при попытке подключения будет отображаться окно с запросом на подтверждение, что обеспечит дополнительную безопасность.

Метод 2: VNC

Альтернативное решение для корпоративных сетей

Что такое VNC?

VNC (Virtual Network Computing) — это метод удаленного доступа к рабочему столу по сети. Данные о действиях специалиста передаются на ваш компьютер через сетевое соединение.

Компьютер специалиста
(VNC Client)
Ваш компьютер
(VNC Server)
Принцип работы:

На вашем компьютере работает VNC-сервер, который обменивается данными с программой VNC-клиент у специалиста поддержки. Изображение вашего экрана передается специалисту, а его действия выполняются на вашем компьютере.

Как узнать IP-адрес компьютера

Что нужно сообщить специалисту:

Для VNC-подключения специалисту необходимо знать IP-адрес вашего компьютера в локальной сети.

Автоматическое отображение

В московских офисах информация о компьютере отображается в правом верхнем углу рабочего стола:

  • Имя компьютера
  • IP-адрес компьютера

Сообщите специалисту эти данные для подключения.

1

Найдите иконку VNC

Дважды кликните по иконке VNC в области уведомлений (правый нижний угол экрана).

Иконка VNC в области уведомлений
1

Откройте командную строку

Нажмите комбинацию клавиш Win + R, затем введите cmd и нажмите Enter.

Окно «Выполнить» для запуска командной строки
2

Выполните команду ipconfig

В командной строке введите ipconfig и нажмите Enter. Найдите строку «IPv4-адрес» — это и есть IP-адрес вашего компьютера.

Вывод команды ipconfig с IP-адресом

Рекомендации по безопасности

Надежные пароли

Используйте сложные пароли с цифрами, буквами и символами. Не сообщайте пароли третьим лицам.

Проверенные специалисты

Предоставляйте доступ только официальным сотрудникам технической поддержки.

Временный доступ

После решения проблемы отключайте неконтролируемый доступ и меняйте пароли.

Контроль сеанса

Следите за действиями специалиста во время удаленного подключения.
