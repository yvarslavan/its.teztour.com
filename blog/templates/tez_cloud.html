Клиенты

CLOUD TEZ TOUR предлагает клиенты под Windows, Android и iOS, что позволяет настроить синхронизацию на всех своих устройствах. С каким бы клиентом вам не пришлось иметь дело, его настройка предельно проста.

Компьютер Windows

1. Установите и запустите клиент с официального сайта
2. Введите адрес сервера https://cloud.teztour.com
3. Введите логин и пароль от учётной записи пользователя ЕРП
4. Выберите локальную папку для синхронизации (например, C:\Nextcloud)
5. Готово! Кинули файл в папку — он улетает на сервер. И наоборот.

Проблемы синхронизации

Файлы не синхронизируются

Проверьте:

  1. Подключение к интернету
  2. Правильность адреса сервера
  3. Актуальность учетных данных
  4. Достаточность места на диске
  5. Статус клиента NextCloud
Конфликты файлов

При конфликтах синхронизации:

  1. Проверьте папку "Конфликты" в NextCloud
  2. Сравните версии файлов
  3. Выберите нужную версию
  4. Удалите конфликтующие файлы

Проблемы загрузки

Медленная загрузка файлов

Оптимизируйте:

  1. Проверьте скорость интернета
  2. Закройте ненужные приложения
  3. Используйте проводное подключение
  4. Проверьте настройки антивируса
Ошибки при загрузке

Устранение ошибок:

  1. Проверьте права доступа к файлу
  2. Убедитесь, что файл не заблокирован
  3. Попробуйте переименовать файл
  4. Очистите кэш браузера

Проблемы доступа

Не удается войти в систему

Проверьте:

  1. Правильность логина и пароля ЕРП
  2. Активность учетной записи
  3. Настройки VPN (если работаете удаленно)
  4. Доступность сервера cloud.teztour.com
Проблемы с общим доступом

Убедитесь:

  1. Файл/папка не заблокированы
  2. Ссылка не истекла
  3. Пароль введен правильно
  4. У вас есть права на общий доступ

Часто задаваемые вопросы (FAQ)

Сколько места доступно в облаке?

Каждому пользователю предоставляется 25 ГБ места в облачном хранилище. При необходимости можно обратиться в IT-поддержку для увеличения лимита.

Можно ли работать с файлами офлайн?

Да, в мобильных приложениях можно скачать файлы для офлайн работы. В десктопном клиенте файлы автоматически синхронизируются при восстановлении соединения.

Как восстановить удаленный файл?

Удаленные файлы попадают в корзину и хранятся там 30 дней. Для восстановления откройте корзину, найдите нужный файл и нажмите "Восстановить".

Можно ли поделиться файлом с внешними пользователями?

Да, можно создать публичную ссылку на файл или папку. Рекомендуется устанавливать срок действия ссылки и пароль для безопасности.

Что делать при конфликте синхронизации?

При конфликте система создает копию файла с пометкой "конфликт". Сравните версии файлов и выберите нужную, затем удалите конфликтующий файл.

Как изменить папку синхронизации в десктопном клиенте?

Откройте настройки клиента NextCloud, перейдите в раздел "Синхронизация" и измените путь к локальной папке. Учтите, что при этом может потребоваться повторная синхронизация.

Поддерживается ли шифрование файлов?

Да, все данные передаются по защищенному SSL/TLS соединению. Для дополнительной защиты рекомендуется шифровать конфиденциальные файлы перед загрузкой.

Можно ли использовать TEZ Cloud на нескольких устройствах?

Да, можно использовать облако на неограниченном количестве устройств. Все изменения будут синхронизироваться между всеми подключенными устройствами.
